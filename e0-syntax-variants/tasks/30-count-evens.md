# 30-count-evens

## Task

Write a beagle function `count-evens` that takes a vector of Longs and
returns the count of even numbers in it. Use `filter` and `count`.

## Expected behavior

`(count-evens [1 2 3 4 5 6])` → `3`
`(count-evens [1 3 5])` → `0`
`(count-evens [])` → `0`
`(count-evens [0])` → `1`
