# 33-my-range

## Task

Write a beagle function `my-range` that takes a Long `n` (non-negative) and
returns a sequence of Longs from `0` to `n-1`. Equivalent to Clojure's
`(range n)` — you may use that internally if you like.

## Expected behavior

`(my-range 0)` → `()` or `[]`
`(my-range 3)` → `(0 1 2)` or `[0 1 2]`
`(my-range 5)` → `(0 1 2 3 4)` or `[0 1 2 3 4]`
