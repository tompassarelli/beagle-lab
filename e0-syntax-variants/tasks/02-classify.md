# 02-classify

## Task

Write a beagle function `classify` that takes a Long and returns a String:

- `"negative"` if the number is less than zero
- `"zero"` if the number is zero
- `"positive"` if the number is greater than zero

## Expected behavior

`(classify -5)` → `"negative"`
`(classify 0)` → `"zero"`
`(classify 7)` → `"positive"`
