# 31-fib

## Task

Write a beagle function `fib` that takes a Long `n` (non-negative) and
returns the `n`-th Fibonacci number.

Fibonacci: `fib(0) = 0`, `fib(1) = 1`, `fib(n) = fib(n-1) + fib(n-2)` for `n >= 2`.

## Expected behavior

`(fib 0)` → `0`
`(fib 1)` → `1`
`(fib 2)` → `1`
`(fib 10)` → `55`
`(fib 15)` → `610`
