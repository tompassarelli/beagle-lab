# 25-cond-many

## Task

Write a beagle function `letter-grade` that takes a Long `score` (0-100)
and returns a String:

- 90 or above → `"A"`
- 80 to 89 → `"B"`
- 70 to 79 → `"C"`
- 60 to 69 → `"D"`
- below 60 → `"F"`

## Expected behavior

`(letter-grade 95)` → `"A"`
`(letter-grade 85)` → `"B"`
`(letter-grade 55)` → `"F"`
